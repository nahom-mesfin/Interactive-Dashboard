# Interactive Dashboard

An interactive, responsive dashboard built with **HTML, CSS, JavaScript, and Chart.js**, featuring animated KPI counters, charts, dark mode, and sidebar navigation. Designed for portfolio projects to showcase front-end skills and interactive UI design.

## Features

- **Responsive Layout:** Works on desktop and mobile screens
- **Animated KPI Counters:** Total sales, new users, and active sessions
- **Charts:**
  - Line chart for sales over time
  - Bar chart for browser usage
  - Doughnut chart for device distribution
- **Dark Mode:** Toggle with persistence using `localStorage`
- **Sidebar Navigation:** Switch between Overview, Analytics, Reports, and Settings
- **Profile & Notifications:** Avatar display and notification badge
- **Date Range Filter:** Update chart data dynamically
- **Update Data Button:** Randomizes chart and KPI data for demo purposes
